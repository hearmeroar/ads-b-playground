# Provenance

These images were downloaded from
[airframesio/airline-images](https://github.com/airframesio/airline-images)
(`flightaware_logos/` subfolder), which itself was generated by running
[Jack Sweeney](https://github.com/Jxck-S)'s
[airline-logo-scraper](https://github.com/Jxck-S/airline-logo-scraper)
against FlightAware. That upstream repository carries no LICENSE file and
states only that the images are "made available for the common good."

Used here strictly for aircraft-operator identification (matching each
image's 3-letter ICAO airline designator filename to a decoded flight
callsign) — a tier-2 fallback only used when
`static/airline-logos/soaring/` (soaring-symbols, MIT) has no logo for that
airline. Every logo remains the trademark/property of its respective
airline; nothing here asserts any ownership over them.
